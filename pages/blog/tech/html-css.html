The Difference Between Relative, Absolute, and Fixed

I'm fixed.
Not even scrolling can get rid of me!
My position is absoulte.
I stick to the page, even if the page moves, and I make room for no one (but my starting position is based off the last element)
I'm relatively positioned.
I make way for others relatively positioned, or they make way for me
I'm relatively positioned too.

Relatively positioned elements will move out of the way of other relatively positioned objects that were declared (made) before them.

Absolutely positioned elements are positioned based off the element before them but they don't care if another element is in the way (also negative margins work).

Fixed elements stick to the screen. No matter how you scroll, fixed elements will always be in the same place.
